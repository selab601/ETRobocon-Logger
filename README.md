# ET Robocon Logger

[ETロボコン](http://www.etrobo.jp/) で使用するために開発したロガーです．
EV3 と Bluetooth 接続すると，機体の状態をリアルタイムで確認できます．
また，ログをファイルに保存し，後に参照することができます．
